# Flight-Delay-Prediction
This project focuses on predicting whether a flight will be delayed by more than 15 minutes using historical U.S. domestic flight data and machine learning techniques. The dataset includes flight operations data from 2013–2018 provided by the Bureau of Transportation Statistics (BTS), along with additional weather and holiday-related features.

The project covers the complete machine learning pipeline, including:

Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Feature engineering
Handling missing values
One-hot encoding categorical variables
Building baseline classification models
Training advanced models using XGBoost
Hyperparameter tuning with Amazon SageMaker
Model evaluation using confusion matrix, ROC-AUC, precision, recall, and F1-score

Additional features such as weather conditions and holiday indicators were incorporated to improve predictive performance.

Project Goal

The goal of this project is to help travel platforms and airlines proactively identify potential flight delays and improve customer travel experience through predictive analytics.
